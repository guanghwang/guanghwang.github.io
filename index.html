Tips to avoid mysterious bugs

Fri, Aug 4, 2017

Don’t use the same for the data.frame and variables in it.

library(data.table)
a <- data.table(a = c(1,2,3,1,2,3), b = c("a", "b", "a", "b","a", "b"))
b <- a[b == "b"]

data.table cannot distinguish which b is. b represents data.table b and the variable b in data.table a.

Notes on the Constant Term in the Fixed Effect Model

Mon, Jul 10, 2017 R notes R
In stata, the fixed effect model(xtreg y x1 x2 x3, fe) report a constant term. The underline estimate is \[y_{it} - \bar{yr} + \bar{\bar{y}} = a + (x_{it} - \bar{x_{i}} + \bar\bar{{x}})) \alpha + (\epsilon_it - \bar{e_i} + \bar{v)\] with the constrant \(\bar{v}\) equals to 0. Details can be found on Stata’s website. In R, plm package does not calculate the some whate artificial1 intercept for within models. see help(“within_intercept”, package = “plm”)↩
